# Rock_paper_scissors
A simple and interactive Rock–Paper–Scissors game built using Python, where a user plays against the computer. The game supports multiple rounds, score tracking, and input validation.

Features
1.Play against the computer
2.Random computer moves using Python’s random module
3.Input validation for user choices
4.Scoreboard with round tracking
5.Replay option after each round
6.Clear game result messages (Win / Lose / Tie)

Technologies Used
- Python 3
- Built-in random module
- Runs in terminal / command line
